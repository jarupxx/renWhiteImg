# renWhiteImg
Install dependencies with ```pip install PIL```
## 概要
読み込んだ画像を真っ白にします.

引数で渡したファイルを全て真っ白します.

白紙ページのゴミを消す時にでも.

## 使用方法

```renWhiteImg.py "C:\path\002.jpg"```

## Summary

This program makes the loaded image white.

All files passed as arguments are white.

Even you want to remove noize on blank pages.

## Usage

```renWhiteImg.py "C:\path\002.jpg"```
